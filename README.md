## 产品简介

Nexus Ones 是一套面向个人用户的工作与生活秩序系统。

它并不是单一的待办软件，也不是单一的日历软件。

而是试图把：

- 日历
- 待办
- 提醒
- 备忘
- 项目
- 课程
- 插件

统一到一个稳定、轻量、可扩展的个人工作台中。

---

## 核心能力

### Today Dashboard

统一查看：

- 今日状态
- 今日提醒
- 今日待办
- 四象限任务
- 项目进度
- 倒计时

### Calendar

支持：

- 年 / 月 / 周 / 日视图
- 中国节假日
- 农历显示
- 日程管理

### Plugin Runtime

v0.2.4 开始正式实装插件运行时。

支持：

- HTML 插件
- JavaScript 插件
- 本地插件存储
- Runtime Sandbox

### Snapshot & Restore

内置快照系统：

- 创建快照
- 恢复快照
- 回滚
- 数据保护

---

## 数据安全

所有用户数据独立于应用本体。

```text
Nexus Ones Data
├── production
├── sandbox
├── plugins
├── backups
└── config
```

支持：

- 覆盖安装不丢数据
- 卸载重装重新识别
- iCloud 跨设备同步
